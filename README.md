# Resume Analyzer & Skill Gap Detector

This project analyzes resumes and identifies missing skills by comparing them with job requirements.

## Features
- Resume skill analysis
- Skill gap detection
- Suggestions for improvement

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (for analysis)

## Purpose
This project helps job seekers understand what skills they need to improve for specific job roles.
